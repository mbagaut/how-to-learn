# Проектная работа №1-2: "Статичная страница"

------
*[Проект](https://mbagaut.github.io/how-to-learn/) выполнен в рамках курса [Яндекс.Практикум "Веб-разработка"](https://praktikum.yandex.ru/web)*
## Описание проекта


Проект позволяет новичку на практике разобраться в методологии БЭМ, и не только следовать одному из популярных принципов именований классов, а именно неймингу по [БЭМ CSS](https://ru.bem.info/methodology/css/), но и изучить сильные стороны классической схемы организации файловой структуры [БЭМ Nested](https://ru.bem.info/methodology/filestructure/#nested).

Добавлен адаптив с использованием media queries и методов резиновой вёрстки.



## Используемые технологии

  * **HTML 5**

  * **CSS 3**
  * **БЭМ**
  * **CSS Flexbox**
  * **Git**
